Introduction

This is is a classic mechanical drawing toy that allows you to create line drawings by turning the boxes in different colors. The default color for the boxes is black.

  • One click turn the color of the box
  • Second click on the colored box erase the color of the box.
  • Reset button clears the board

Have fun with drawing

Etch-A-Sketch

Grid Size: 16x16

Drawing

You can also use the drawing function instead of clicking individual boxes. Simply click the "Draw" button to activate it. You can deactivate this function at any time as well. First click will start and the second click will pause the drawing. To be able to use the random color black again just refresh the page.

When you click the "Randomize" button, it will automatically activate the drawing function, and each box will be assigned a different color.
